How to Solve Admin Console Credential Issue

Posted January 06, 2018

Informatica 10 has known issue with it's new admin console which asks for an username and password inside admin console even after successful login.

Credential Issue in Informatica 10 Admin Console

  • Login admin console with Administrator privileged account.
  • After successful login you will get one more login prompt which would block some of the dashboard views.

    Even after providing correct credentials again in the blocker you would throw an error message for an Invalid Credentials or there would be no post request for submitted username and password.

Solution

This is a case of broken admin console in Informaitca 10. To resolve admin console credential issue you should build admin console again by doing tomcat cleanup.
